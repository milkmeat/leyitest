# 游戏界面说明

本文档描述冰封之岛（Frozen Island Pro）的游戏界面元素，供开发任务参考。

## 任务(quest)提示栏

- 主城界面才有任务提示栏
- 在屏幕下部，聊天信息框的上方，是任务提示栏。
- 任务提示栏左端是"卷轴"图标（右上角有红色数字提示表示有已完成未领取奖励的任务），点击会打开任务列表。通常分为 章节任务、成长任务、每日任务
- "卷轴"图标右侧的文本是"当前任务"，点击会直接跳转到任务执行入口。
- 在新手指引过程中，常常会有一个手指图标指向当前任务
- 当前任务完成后，会在任务文本右边出现一个绿色对勾，表示可以领取奖励
- 如果有未领取的奖励，先把当前奖励全部领取再继续后面的任务

## 建筑升级
- 在建筑升级界面，需要点击上方的外部空白区域退出
- 升级建筑时，需要先升级家具
- 建筑名称栏右侧有一个进度条，需要先升级若干个家具（每个家具要点若干下）使其达到100%才会出现升级按钮
- 在这个进度条下方有一个较大的"建造"/"升级"/"下一个"按钮用于升级家具,可以连续点击增加进度。点击一次大约增加10%进度，你可以估算需要连续点击的次数。该按钮可以连续点击，出现"下一个"后会自动跳到下一家具，不需要选择家具。
- 进度条达到100%后，进度条消失，点击蓝色"升级"按钮，才可以升级建筑。
- 如果""确认"/"升级"按钮上显示一个时间（如00:01:30），表示建造需要的时间，要点一下才会开始建造。
- 建筑开始升级后，建筑中央会出现倒数计时，需要等待其完成。
- 点击对话框上方的空白部分，可以退出升级对话框

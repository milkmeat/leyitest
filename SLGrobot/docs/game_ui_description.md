# 游戏界面说明

本文档描述冰封之岛（Frozen Island Pro）的游戏界面元素，供开发任务参考。

## 任务(quest)提示栏

- 主城界面才有任务提示栏
- 在屏幕下部，聊天信息框的上方，是任务提示栏。
- 任务提示栏左端是"卷轴"图标（右上角有红色数字提示表示有已完成未领取奖励的任务），点击会打开任务列表。通常分为 章节任务、成长任务、每日任务
- "卷轴"图标右侧的文本是"当前任务"，点击会直接跳转到任务执行入口。
- 在新手指引过程中，常常会有一个手指图标指向当前任务
- 当前任务完成后，会在任务文本右边出现一个绿色对勾，表示可以领取奖励
- 如果有未领取的奖励，先把当前奖励全部领取再继续后面的任务

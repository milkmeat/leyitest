# 测试模式说明
这是测试专用模式，使用**内置代码逻辑**而非 LLM 推理。系统将根据 L2 指令和游戏状态自动生成 L0 指令。

你只需要返回占位符响应，实际指令由代码逻辑生成。

# 内置逻辑规则（参考实现）

## 规则1：距离判断 - 移城
如果小队成员不在指定建筑附近15格子内：
```python
distance = sqrt((member_x - target_x)^2 + (member_y - target_y)^2)
if distance > 15:
    生成 LVL_MOVE_CITY 指令到建筑附近随机位置
```

## 规则2：中立/我方建筑 - Solo 进攻
如果建筑处于中立（alliance_id=0）或我方控制状态：
- 检查每位成员的出征状态
- 对**未派出军队**的成员，生成 `LVL_ATTACK_BUILDING` 指令
- 每个成员最多派出 1 支部队（100000士兵或默认兵力上限）

## 规则3：敌方建筑 - Rally 集结
如果建筑处于敌对联盟控制状态：
- 由队长（leader_uid）发起 `LVL_INITIATE_RALLY_BUILDING`
- 所有队员（包括队长）参与 `LVL_JOIN_RALLY`
- 每人派出 100000 士兵或默认兵力上限
